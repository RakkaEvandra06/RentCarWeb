## DriftDreams
Is a comprehensive car rental web application built with PHP, MySQL, and Tailwind CSS. The platform allows users to browse available vehicles, view detailed specifications, make rental bookings, and interact with the community through comments and testimonials.

## 🚀 Usage
### 👤 For Users
Browse Vehicles: Navigate to "Our Car" section to view available vehicles
View Details: Click on any vehicle to see specifications
Create Account: Sign up with email and password
Login: Access your account
Make Booking: Select rental package and proceed to payment
Leave Comments: Share your experience in the comment section
Contact: Use contact form for inquiries

### 👥 For Administrators
Monitor user registrations in sign_up table
Review and moderate comments in posts table
Update vehicle information in portfolio pages
Respond to contact form submissions

## ✨ Features
User Authentication
Sign Up: New users can create accounts with email and password
Login: Secure authentication for returning users
Password Reset: Forgot password functionality for account recovery

## 💡 Additional Features
Newsletter subscription
Social media integration
Responsive design for mobile and desktop
24/7 customer support highlighted

## 🔒 Security Features
Password storage (currently plain text - should be hashed in production)
MySQL real escape string for SQL injection prevention
Form validation on client side
Session-based authentication
